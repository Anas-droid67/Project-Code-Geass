Geass Users

Alle bekende Geass-gebruikers uit de originele Code Geass-serie.

Lelouch vi Britannia

Geass: Absolute Obedience

Type: Mind Control

Contract: C.C.

Verbannen prins van Britannia en leider van de Black Knights.

Kan via oogcontact een absoluut bevel geven dat niet geweigerd kan worden.

Bekend van de Black Rebellion en de Zero Requiem.

Mao

Geass: Mind Reading

Type: Telepathie

Contract: C.C.

Een voormalige bondgenoot van C.C.

Kan de gedachten van mensen lezen binnen een grote straal.

Zijn Geass groeide zo sterk dat hij hem niet meer kon uitschakelen.

Rolo Lamperouge

Geass: Time Perception Stop

Type: Waarnemingscontrole

Bereik: Omgeving

Lid van een Geass-programma van Britannia.

Kan de tijdswaarneming van mensen tijdelijk stilzetten.

Veelvuldig gebruikt voor moorden en infiltraties.

Charles zi Britannia

Geass: Memory Rewrite

Type: Geheugenmanipulatie

Bereik: Oogcontact

De 98e keizer van Britannia.

Kon herinneringen wissen, veranderen of volledig herschrijven.

Gebruikte zijn Geass op Lelouch en Nunnally.

Marianne vi Britannia

Geass: Mind Transfer

Type: Bewustzijnsoverdracht

Bereik: Persoonlijk

Bekend als de Flashing Duchess.

Kon haar bewustzijn overbrengen naar een ander lichaam.

Hierdoor overleefde ze haar moordaanslag.

Bismarck Waldstein

Geass: Future Sight

Type: Precognitie

Bereik: Zelf

De Knight of One en sterkste ridder van Britannia.

Kon enkele seconden vooruit kijken in de toekomst.

Maakte hem bijna onverslaanbaar in gevechten.

C.C.

Status: Code Bearer

Voormalige Geass: Liefde

Contractgever: Ja

Oorspronkelijk zelf een Geass-gebruiker.

Haar Geass zorgde ervoor dat iedereen van haar hield.

Na het verkrijgen van een Code verloor zij haar Geass.

V.V.

Status: Code Bearer

Voormalige Geass: Onbekend

Contractgever: Ja

Tweelingbroer van Marianne's contractpartner.

Bezat een Code en schonk Charles zijn Geass.

Zijn oorspronkelijke Geass is nooit onthuld.
